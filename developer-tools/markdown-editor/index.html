Markdown Cheat Sheet

  • **bold** or __bold__ — bold text
  • *italic* or _italic_ — italic text
  • `code` — inline code
  • ## Heading — H2 heading (use 1-6 #s)
  • - item — unordered list
  • [text](url) — hyperlink
