Free Drag & Drop Form Builder

Build HTML forms visually without writing code. Drag fields from the palette, configure labels, placeholders, validation rules, and options in the properties panel, then copy the generated HTML in Bootstrap 5, Tailwind CSS, or vanilla CSS styles.

Supported Field Types

  • Text, email, password, number, phone, URL inputs
  • Textarea, date/time pickers, file upload, range slider
  • Select dropdowns, radio groups, checkboxes, checkbox groups
  • Hidden fields, headings, paragraph text, dividers, submit buttons

CSS Modes

Choose Bootstrap 5 to get Bootstrap component classes ready to use with the Bootstrap CDN. Choose Tailwind CSS for utility-class output compatible with Tailwind v3. Choose Vanilla CSS for self-contained output with a <style> block.
