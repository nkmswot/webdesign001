Strong Password Generator

A strong password should be at least 16 characters and include uppercase letters, lowercase letters, numbers, and symbols. This generator uses the browser's crypto.getRandomValues() API for cryptographically secure randomness — far stronger than Math.random().

Password Security Tips

  • Use a unique password for every account
  • Use a password manager (Bitwarden, 1Password, etc.) to store passwords
  • Enable two-factor authentication wherever possible
  • A 20-character random password takes billions of years to brute-force
