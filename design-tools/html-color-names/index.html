All 147 HTML Color Names

CSS and HTML support 147 named colors defined in the CSS Color Level 4 specification. These range from common colors like red and blue to descriptive names like PapayaWhip, MidnightBlue, and LemonChiffon.

How to Use Named Colors in CSS

Any CSS property that accepts a color value can use a named color: color: tomato, background: steelblue, border: 2px solid coral. Named colors are case-insensitive.
