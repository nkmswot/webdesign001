CSS Animation Generator

CSS animations use the @keyframes rule to define animation states. This generator covers all animation-* properties: duration, delay, iteration, easing, direction, and fill mode.

CSS Animation Properties

  • animation-duration — how long one cycle takes
  • animation-timing-function — the easing curve (ease, cubic-bezier, steps)
  • animation-iteration-count — how many times to play (or infinite)
  • animation-fill-mode: forwards — keeps the end state after animation finishes
  • animation-direction: alternate — plays forwards then backwards
